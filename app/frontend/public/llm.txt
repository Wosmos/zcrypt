# zpush

zpush is a zero-knowledge encrypted personal cloud storage and file sharing platform.

## Key Features
- **Client-Side Encryption:** Files are compressed and encrypted (AES-256-GCM) directly in the browser before being uploaded. The server never sees the raw data or the keys.
- **Zero-Knowledge Architecture:** Since keys are derived locally and never transmitted, zpush is completely mathematically private.
- **Lightning Fast Uploads:** Real-time chunking and streaming directly to cloud storage natively inside the application.

## Tech Stack
- Frontend: Next.js (React), Tailwind CSS, Framer Motion for beautiful interactions and animations.
- Cryptography: Web Crypto API for secure and fast client-side symmetric encryption.
